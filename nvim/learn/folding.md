# how to folding in vim

## VISUAL MODE

- `zf{motion}[number]`: create fold base on motion -> it gonna folding all selected
- `zo`: open it
- `za`: open all folding
- `zc`: will close it
- `zr`: decreases the foldlevel by one.
- `zm`: increases the foldlevel by one.
- `zR`: decreases the foldlevel to zero.
